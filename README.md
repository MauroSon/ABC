# ABC - Arquivos de Texto

Este repositório contém uma coleção de arquivos de texto simples, cada um contendo uma única letra correspondente ao seu nome.

## Arquivos

O projeto consiste em seis arquivos de texto localizados na raiz do repositório:

| Arquivo | Conteúdo | Descrição |
|---------|----------|-----------|
| `A` | A | Arquivo contendo a letra "A" |
| `B` | B | Arquivo contendo a letra "B" |
| `C` | C | Arquivo contendo a letra "C" |
| `D` | D | Arquivo contendo a letra "D" |
| `E` | E | Arquivo contendo a letra "E" |
| `F` | F | Arquivo contendo a letra "F" |

## Propósito

Estes arquivos servem como uma estrutura simples e sequencial, representando as primeiras seis letras do alfabeto. Cada arquivo é independente e contém apenas seu caractere identificador correspondente.

## Estrutura

```
.
├── A         # Arquivo de texto com conteúdo "A"
├── B         # Arquivo de texto com conteúdo "B"
├── C         # Arquivo de texto com conteúdo "C"
├── D         # Arquivo de texto com conteúdo "D"
├── E         # Arquivo de texto com conteúdo "E"
├── F         # Arquivo de texto com conteúdo "F"
└── README.md # Esta documentação
```

## Como Utilizar

Para visualizar o conteúdo de qualquer arquivo, use um dos comandos abaixo:

```bash
# Usando cat
cat A

# Usando less
less B

# Usando um editor de texto
nano C
```
