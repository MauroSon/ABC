# ABC - Repositório Experimental

## Descrição

Repositório experimental utilizado para testes com Claude Code e fluxos de trabalho Git. Este projeto contém arquivos simples de teste para validar processos de desenvolvimento e integração contínua.

## Propósito

Este repositório serve como ambiente controlado para:
- Testar funcionalidades do Claude Code
- Validar fluxos de trabalho Git
- Experimentar com branches e merges
- Desenvolver e refinar processos de CI/CD

O projeto utiliza uma estrutura de branches experimental para desenvolvimento:

- **main**: Branch principal com código estável
- **synkrony/setup-***: Branches de feature para experimentação

## Como Usar

### Clonar o Repositório

```bash
git clone https://github.com/MauroSon/ABC.git
cd ABC
```

### Navegar pelos Arquivos

```bash
# Ver conteúdo dos arquivos
cat A B C D E F

# Listar todos os arquivos
ls -la
```

### Verificar Versão

```bash
cat VERSION.md
```
